# API Reference: GapCheck

This document provides a comprehensive guide to the serverless backend API endpoints exposed by GapCheck.

---

## 1. File Parsing Endpoint

### `POST /api/parse-file`

Extracts plain text from document binaries. This endpoint runs on the standard Node.js serverless runtime.

#### Request Headers
- `Content-Type: multipart/form-data`

#### Request Payload
The body must be formatted as `multipart/form-data` containing:
- `file` (Binary File): The PDF (`.pdf`) or Word (`.docx`) file to parse.

#### Constraints & Validation
- **Allowed Extensions**: `.pdf`, `.docx` (case-insensitive checks)
- **Maximum File Size**: 5 MB ($5,242,880$ bytes)
- **Minimum Output Length**: 30 characters (prevents scanned image uploads or blank file successes)

#### Successful Response (`200 OK`)
Returns extracted plain text, truncated to $6,000$ characters if it exceeds client caps.

```json
{
  "text": "John Doe\nSoftware Engineer\nExperience:\n- Built React websites..."
}
```

#### Error Responses

- **`400 Bad Request`** (Invalid boundary/mime-type)
  ```json
  { "error": "Invalid request — expected multipart/form-data." }
  ```
- **`400 Bad Request`** (Missing file parameter)
  ```json
  { "error": "No file received. Please select a PDF or DOCX file." }
  ```
- **`400 Bad Request`** (Unsupported extension type)
  ```json
  { "error": "Unsupported file type \".png\". Please upload a PDF (.pdf) or Word document (.docx)." }
  ```
- **`400 Bad Request`** (File too large)
  ```json
  { "error": "File too large (6.2 MB). Please upload a file under 5 MB, or paste the text directly." }
  ```
- **`422 Unprocessable Entity`** (Empty or image-only PDF)
  ```json
  { "error": "This PDF appears to contain no selectable text — it may be a scanned image. Please paste the text manually instead." }
  ```
- **`422 Unprocessable Entity`** (Corrupt document)
  ```json
  { "error": "Could not extract text from this PDF. It may be a scanned image without selectable text, or the file may be corrupted." }
  ```
- **`500 Internal Server Error`** (Unexpected execution fault)
  ```json
  { "error": "An unexpected error occurred while parsing the file. Please try again." }
  ```

## 2. Analysis Endpoint

### `POST /api/analyze`

Performs JD vs. Resume comparison using Google Gemini.

#### Request Headers
- `Content-Type: application/json`

#### Request Payload
```json
{
  "jdText": "The job description plain text...",
  "resumeText": "The resume plain text..."
}
```

#### Constraints & Validation
- Both inputs must be non-empty strings.
- **Minimum Input Length**: 100 characters each (trimmed).
- **Maximum Input Length**: 6,000 characters each.

#### Successful Response (`200 OK`)
Returns the structured JSON analysis payload generated by the LLM.

```json
{
  "matchScore": 76,
  "matchedSkills": [
    "React",
    "Tailwind CSS",
    "Node.js"
  ],
  "missingSkills": [
    {
      "skill": "Next.js",
      "importance": "critical",
      "reason": "The JD lists Next.js as the core application framework and requires experience building server-rendered applications."
    },
    {
      "skill": "TypeScript",
      "importance": "important",
      "reason": "TypeScript is preferred for code quality and is specified in the requirements."
    },
    {
      "skill": "GraphQL",
      "importance": "nice-to-have",
      "reason": "Mentioned as a preferred tool for data fetching, but isn't a core requirement."
    }
  ],
  "bulletRewrites": [
    {
      "before": "Updated company websites and code.",
      "after": "Refactored core modules to React and Next.js, increasing responsive loading times by 30% and matching critical job specifications."
    }
  ]
}
```

#### Error Responses

- **`400 Bad Request`** (Invalid format)
  ```json
  { "error": "Invalid request body. Expected JSON." }
  ```
- **`400 Bad Request`** (Short input)
  ```json
  { "error": "Job description is too short. Please paste the full job description (minimum 100 characters)." }
  ```
- **`400 Bad Request`** (Long input)
  ```json
  { "error": "Resume is too long. Please limit to 6,000 characters." }
  ```
- **`400 Bad Request`** (AI Safety block)
  ```json
  { "error": "The content was blocked by Gemini safety filters. Please review your input." }
  ```
- **`429 Too Many Requests`** (Gemini Rate Limits)
  ```json
  { "error": "Gemini API rate limit reached. Please wait a moment and try again." }
  ```
- **`500 Internal Server Error`** (Missing server-side configuration key)
  ```json
  { "error": "Server configuration error. The Gemini API key is not configured." }
  ```
- **`500 Internal Server Error`** (Incorrect key / authentication error)
  ```json
  { "error": "Invalid Gemini API key. Please check your configuration." }
  ```
- **`500 Internal Server Error`** (AI returned invalid schema structure)
  ```json
  { "error": "The AI returned an incomplete analysis. Please try again." }
  ```
- **`502 Bad Gateway`** (AI model response generation timeout)
  ```json
  { "error": "Failed to get a response from the AI model. Please try again." }
  ```
